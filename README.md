# Quiz-website
This is a web-based quiz website that allows users to answer questions and view their score at the end of the quiz.

## Features
- Instant score calculation
- User-friendly interface
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Clone or download the repository
2. Open `index.html` in any web browser
3. Start the quiz and answer the questions
4. View your final score

## Project Structure
- index.html – Main quiz page
- style.css – Styling of the website
- script.js – Quiz logic and functionality
